Desktop Engineering Releases

 

This page catalogues releases from desktop engineering teams. Its content automatically generated every 15 minutes by querying the Github API and scraping Launchpad pages. The data is generated using releasegen.

  • For a release to show here, it must be part of one of the specified Github Teams or Launchpad project groups. Github projects must not be Archived, Launchpad projects are scraped for tags. The latest three releases are shown for each repository.

    For GitHub badges to be shown for your repository, add them in the README.
